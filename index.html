What are in this repository?

This repository contains some small experiments I did with bagging algorithm and K-mean algorithm. Besides, it also contains the code for a public machine learning contest.

The contest can be found here

What did I do in the contest code?

Basically, to predict whether or not the water supply is polluted, I learned part of the public data sets (training set) using Logistic Regression model, Decision Tree model (bagging and random forest). Then, test the models using the test set separated from the overall data set. Once I know which model performs better, I will train the model again using all the data sets I have. Eventually, I used the final model for predicting the official test set whose result is to be submitted to the contest.
